# ①課題名
じゃんけんアプリ

## ②課題内容（どんな作品か）
- 3回じゃんけんをして、その合計点が加算されていくアプリです。

## ③アプリのデプロイURL
https://mitsuko117.github.io/kadai01_janken/

## ④アプリのログイン用IDまたはPassword（ある場合）
なし

## ⑤工夫した点・こだわった点
- 点数が加算されるようにしました。
- 勝ち、負けの表示を2秒で消えるようにしました。
- 回数0になり、じゃんけんのアイコンを押すと、GAMEOVERが流れます。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 条件分岐で頭がごちゃごちゃになり、理解が追いつかない部分がありました。
- GAMEOVERを表示した後、以前の画面でリセットボタンを押せるようにすること
- 次回はランキング表示を実装したり、見た目をもっと美しく整えたいです。
